# Solar Tracking Device

This repository contains the version-controlled code for my solar tracking project.

The project uses LDR sensors to detect light intensity and servo motors to adjust the position of a solar panel or sensor platform. The aim is to improve light capture by automatically orientating the system toward the strongest light source.

Three scripts were developed during the project:

1. Basic LDR tracking script
2. Dual-axis tracking script
3. Final Arduino Nano ESP32 script with improved pin mapping and comments

GitHub was used to manage changes, create branches, document improvements, and track the development of the code.
